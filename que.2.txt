Que.2 Write a function to find whether the given input number is Odd. If the given number is odd, the function should return 2 else it should return 1.
Note: The number passed to the function can either be negative, positive or zero. Zero should not be treated as odd.

Ans.2 

import java.io.*;
import java.util.*;
class UserMainCode
{
public int isOdd(int input1){
if(input1 %2!=0){
return 2;
}else{
return 1;
}
}
}
