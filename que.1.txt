Que.1  Write a function to find whether the given input number is Even. If the given number is even, the function should return 2 else it should return 1. 
Note: The number passed to the function can either be negative, positive or zero. Zero should be treated as Even.
Ans.1

import java.io.*;
import java.util.*;
class UserMainCode{
public int isEven(int input1){
if(input1 %2==0){
return 2;
}else{
return 1;
}
}
}
